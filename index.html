API name API description
  1. Find nearby populated place / reverse geocoding
 returns the closest populated place (feature class=P) for the lat/lng query as xml document. The unit of the distance element is 'km'. This service is also available in JSON format.
  1. CountryCode / reverse geocoding
 returns the iso country code for the given latitude/longitude
  1. Elevation - GTOPO30
 a single number giving the elevation in meters according to gtopo30, ocean areas have been masked as "no data" and have been assigned a value of -9999
